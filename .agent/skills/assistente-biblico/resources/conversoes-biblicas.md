# Tabela de Conversões Bíblicas

Referência rápida para conversão de pesos, medidas, moedas e tempo usados na Bíblia.

## Pesos

| Termo Bíblico | Equivalência Aproximada |
|---------------|------------------------|
| Siclo         | ~11,4 gramas           |
| Mina          | ~571 gramas (50 siclos)|
| Talento       | ~34 kg (60 minas)      |
| Beca          | ~5,7 gramas (½ siclo)  |
| Gera          | ~0,57 gramas (1/20 siclo) |
| Libra (NT)    | ~327 gramas            |

## Moedas

| Termo Bíblico  | Equivalência Funcional                |
|----------------|--------------------------------------|
| Denário        | 1 dia de trabalho de um operário     |
| Dracma         | Similar ao denário                   |
| Siclo de prata | ~4 denários                          |
| Talento (moeda)| ~6.000 denários (~20 anos de trabalho)|
| Lepto (viúva)  | Menor moeda em circulação            |
| Asse           | 1/16 de denário                      |
| Quadrante      | 1/64 de denário                      |
| Estáter        | ~4 dracmas (ou 1 siclo)             |

## Tempo / Horários

O dia judaico era dividido em "horas" a partir do nascer do sol (~6h):

| Hora Bíblica     | Hora Moderna Aproximada |
|-------------------|------------------------|
| Hora primeira     | ~6h da manhã           |
| Hora terceira     | ~9h da manhã           |
| Hora sexta        | ~12h (meio-dia)        |
| Hora nona         | ~3h da tarde           |
| Hora undécima     | ~5h da tarde           |

### Vigílias da Noite

| Segunda vigília   | 21h–00h               |
| Terceira vigília  | 00h–3h                |
